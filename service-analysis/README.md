# 🔍 service-analysis

서비스·플랫폼·앱을 PM·전략 관점에서 5단계로 종합 분석하는 프레임워크.

## 트리거

```
"○○ 서비스 분석해줘"
"○○ 어떤 서비스야?"
"경쟁사 분석해줘"
"○○ 사이트 분석"
URL 붙여넣기
```

## 분석 구조

```
STEP 0. 비즈니스 구조도 (플랫폼 구조 시각화)
STEP 1. 서비스 현황
        └ 히스토리 타임라인
        └ 시장 파이차트 + 업계 현황
        └ 경쟁사 3개 심층 비교 (레이더차트 + 비교표)
STEP 2. 사용자층 분석
        └ 2x2 매트릭스 + 세그먼트 카드
STEP 3. 서비스 페이지 분석
STEP 4. 수익 구조 분석
        └ 수익·성장 그래프
STEP 5. 인사이트 & 제안
        └ AI 활용 현황 체크
        └ 3단계 로드맵
```
